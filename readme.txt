Search & Replace in your posts and pages

== Description ==

This Plugin allows you to search and replace in the filters the_content, the_excerpt, the_title

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Manage > Text-Replacement
1. Add the replacement rules

== Frequently Asked Questions ==

= Why should I use this plugin? = 

This plugin looks stupid but is very powerfull, you could replace the names of companys with their logo or simply link to WordPress if you write the word WordPress in your posts and pages. Visit the Pluginsite!
